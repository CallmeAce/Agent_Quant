**验收标准:**
- [ ] 每个因子能独立计算并输出结果
- [ ] 因子结果正确写入 dwd_factor_value 表
- [ ] 因子值范围合理（无异常值）
- [ ] 支持按日期、股票范围批量计算

**因子计算公式参考:**
```
PE_TTM = 总市值 / 过去 12 个月归母净利润
PB = 总市值 / 净资产
ROE = 归母净利润 / 平均净资产
PS_TTM = 总市值 / 过去 12 个月营业收入
Debt_to_Equity = 总负债 / 净资产
```

**验收标准:**
- [ ] CN_VALUE_MID_V1 策略能独立运行
- [ ] 输出选股结果（股票列表 + 权重）
- [ ] 符合策略文档中的约束条件
- [ ] 支持回测模式和实盘模式切换

**CN_VALUE_MID_V1 选股流程:**
```
1. 标的池过滤
   └─ 剔除 ST/*ST
   └─ 剔除近 3 个月日均成交额 < 2000 万
   └─ 剔除股价 < 2 元

2. 因子打分
   └─ PE_TTM (30% 权重)
   └─ PB (25% 权重)
   └─ ROE (25% 权重)
   └─ PS_TTM (10% 权重)
   └─ Debt_to_Equity (10% 权重)

3. 排序与筛选
   └─ 综合得分排序
   └─ 选取 Top 30-50
   └─ 行业中性约束 (单行业≤20%)

4. 权重分配
   └─ 单票最大 5%，最小 1%
   └─ 等权重或得分加权
```
